How to Use MiniShare

How do I share a big amount of files easily?

 Just drag and drop a directory. But remember that you have to do that again if you add files to the directory etc. Also, you can't list the contents of the directory, you always must access exact file in the directory.

How can I keep the same IP so the links work always?

 With dynamic IPs (such as dial-ups) you can't get the same IP every time. But you can get a free domain name from serveral places. After you have a domain name, put the domain name in the "Fixed domain name" field in Confing. Remember, that the IP still changes so you have to update the IP in the domain name service's settings. They have more info about this for sure.

What are those random letters?

 If you share lots of files you might want to restrict people from using your bandwidth. So, if you send an URL to someone to let him/her get one file from your computer, he/she could edit the URL so if he/she knows a filename of an another file you sharedr, he'd be able to get it. But when there's a lot of purely random letters before the filename, he/she can't guess all of them, resulting to the infamous 404.

Troubleshooting

Can't bind server.

 As the error message says, you have to change the port in the Config dialog. The port was most likely in use by another program.

MiniShare can't get the (right) IP address for my computer.

 You have to enter the proper IP in the Config dialog.
